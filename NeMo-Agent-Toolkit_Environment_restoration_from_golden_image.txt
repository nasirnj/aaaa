## Option 2 ‚Äì ‚ÄúInstaller bundle‚Äù (rebuild venv from wheels)

You already have a nice portable installer **inside** `nemo-agent-toolkit`:

* `wheels/` ‚Äì all wheels (including `nvidia-nat*`, `pip`, `wheel`, etc.)
* `requirements_offline.txt` ‚Äì all pinned deps

If you ever want to rebuild the venv fresh on the new air-gapped box:

### On the *target* box (after extracting the tarball)

```bash
cd /root/nemo-agent-toolkit

# 1) Create a fresh venv with Python 3.12
python3.12 -m venv .venv
source .venv/bin/activate

# 2) Install the NeMo wheels first (no deps, from local dir)
pip install --no-index --find-links=wheels --no-deps \
  nvidia-nat nvidia-nat-langchain nvidia-nat-test

# 3) Install the rest from the offline requirements
pip install --no-index --find-links=wheels -r requirements_offline.txt

# 4) (Optional) Satisfy nvidia-nat's pip requirement
pip install --no-index --find-links=wheels "pip>=24.3.1"

# 5) Test
nat run --config_file workflow.yml --input "hello from rebuilt env"
```

This gives you a **reproducible install** even if the venv gets broken or you change Python minor versions later.
